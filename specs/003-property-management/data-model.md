# Data Model Design: Property Management

## Entities

### 1. Property
Represents a real estate listing.

| Field | Type | Nullable | Validation / Constraints |
|---|---|---|---|
| **Id** | Guid | No | Primary Key |
| **Title** | string | No | Max length 200, Required (except in early Draft) |
| **Description** | string | Yes | Max length 4000 |
| **ShortDescription** | string | Yes | Max length 500 |
| **Price** | decimal | No | Must be >= 0 |
| **ListingType** | enum | No | `Sale`, `Rent` |
| **CategoryId** | Guid | Yes | Foreign Key to Categories |
| **PropertyTypeId** | Guid | Yes | Foreign Key to PropertyTypes |
| **StatusId** | Guid | Yes | Foreign Key to ListingStatuses |
| **ConditionId** | Guid | Yes | Foreign Key to Conditions |
| **Bedrooms** | int | Yes | Must be >= 0 |
| **Bathrooms** | int | Yes | Must be >= 0 |
| **Balconies** | int | Yes | Must be >= 0 |
| **Floors** | int | Yes | Must be >= 0 |
| **Parking** | int | Yes | Must be >= 0 |
| **Area** | decimal | Yes | Must be >= 0 |
| **AreaUnit** | string | Yes | e.g. "sqft", "sqm" |
| **LotSize** | decimal | Yes | Must be >= 0 |
| **FurnishedStatus** | string | Yes | e.g. "Furnished", "Unfurnished" |
| **YearBuilt** | int | Yes | Must be > 1800 |
| **FacingDirection** | string | Yes | e.g. "North", "East" |
| **CountryId** | Guid | Yes | Foreign Key to Countries |
| **StateId** | Guid | Yes | Foreign Key to States |
| **CityId** | Guid | Yes | Foreign Key to Cities |
| **AreaText** | string | Yes | Free text field for Area |
| **Address** | string | Yes | Max length 500 |
| **Landmark** | string | Yes | Max length 200 |
| **ZipCode** | string | Yes | Max length 20 |
| **Latitude** | double | Yes | Between -90 and 90 |
| **Longitude** | double | Yes | Between -180 and 180 |
| **Slug** | string | Yes | Unique index, URL friendly |
| **MetaTitle** | string | Yes | Max length 100 |
| **MetaDescription** | string | Yes | Max length 160 |
| **MetaKeywords** | string | Yes | Max length 200 |
| **OwnerId** | string | No | Foreign Key to AspNetUsers |
| **PublishStatus** | enum | No | `Draft`, `PendingApproval`, `Published`, `Rejected`, `Archived` |
| **CreatedDate** | DateTime | No | Autogenerated |

### 2. PropertyMedia
Media items associated with a listing.

| Field | Type | Nullable | Validation / Constraints |
|---|---|---|---|
| **Id** | Guid | No | Primary Key |
| **PropertyId** | Guid | No | Foreign Key to Property |
| **FilePath** | string | No | File system path |
| **FileType** | string | No | "Image" or "Video" |
| **IsFeatured** | bool | No | Default `false` |
| **DisplayOrder** | int | No | Default `0` |

### 3. PropertyDocument
Documents associated with a listing.

| Field | Type | Nullable | Validation / Constraints |
|---|---|---|---|
| **Id** | Guid | No | Primary Key |
| **PropertyId** | Guid | No | Foreign Key to Property |
| **FilePath** | string | No | File system path |
| **DisplayName** | string | No | Maximum length 100 |
| **IsPublic** | bool | No | Default `true` |

### 4. PropertyFloorPlan
Floor plans showing dimensions and shapes.

| Field | Type | Nullable | Validation / Constraints |
|---|---|---|---|
| **Id** | Guid | No | Primary Key |
| **PropertyId** | Guid | No | Foreign Key to Property |
| **FilePath** | string | No | File system path |
| **Name** | string | No | e.g. "Ground Floor" |
| **Dimensions** | string | Yes | e.g. "40 x 50 ft" |

### 5. PropertyAuditLog
History of listings status updates.

| Field | Type | Nullable | Validation / Constraints |
|---|---|---|---|
| **Id** | Guid | No | Primary Key |
| **PropertyId** | Guid | No | Foreign Key to Property |
| **UserId** | string | No | Action performer ID |
| **OldStatus** | enum | No | Status before transition |
| **NewStatus** | enum | No | Status after transition |
| **Notes** | string | Yes | Moderator notes / comments |
| **CreatedDate** | DateTime | No | Timestamp |

## Relationships

- **Property** $\rightarrow$ `has many` **PropertyMedia** (cascade delete)
- **Property** $\rightarrow$ `has many` **PropertyDocument** (cascade delete)
- **Property** $\rightarrow$ `has many` **PropertyFloorPlan** (cascade delete)
- **Property** $\rightarrow$ `has many` **PropertyAuditLog** (cascade delete)
